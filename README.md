## Descripción del proyecto

MercApp es una aplicación web de una sola página desarrollada con Vue 3 y Vite, conectada a una API REST propia construida con Node.js y Express.

El objetivo del proyecto es implementar un catálogo de productos con búsqueda, filtro por categoría, detalle de producto, formulario de creación y edición, manejo de carrito simple y consumo asíncrono de datos desde un backend propio.

## Tecnologías utilizadas

- API REST

### Frontend
