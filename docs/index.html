Resumen técnico del sistema

MercApp es una aplicación web tipo SPA desarrollada con Vue 3 para la gestión de productos y categorías. El sistema cuenta con un backend propio construido con Node.js y Express, conectado a una base de datos MongoDB Atlas.

En la Unidad 4 se realizó el despliegue completo del sistema usando servicios en la nube: MongoDB Atlas para la base de datos, Railway para la API, Netlify para el frontend y GitHub Pages para este micrositio.

Arquitectura de despliegue

  • Frontend: Vue 3 + Vite publicado en Netlify.
  • Backend: Node.js + Express publicado en Railway.
  • Base de datos: MongoDB Atlas, base mercapp.
  • Repositorio: GitHub.
  • Micrositio: GitHub Pages.

Endpoints principales

Método Ruta Descripción
GET /health Verifica el estado de la API y la conexión general del servicio.
GET /api/products Lista todos los productos registrados.
GET /api/products/:id Consulta un producto por su identificador.
POST /api/products Crea un nuevo producto.
PUT / PATCH /api/products/:id Actualiza un producto existente.
DELETE /api/products/:id Elimina un producto.
GET /api/categories Lista todas las categorías registradas.
POST /api/categories Crea una nueva categoría.

Guía breve de ejecución local

Backend

Crear un archivo backend/.env basado en backend/.env.example.

cd backend
npm install
npm run dev

Frontend

Crear un archivo de entorno con la variable VITE_API_URL.

cd frontend
npm install
npm run dev
